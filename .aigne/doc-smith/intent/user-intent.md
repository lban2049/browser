# User Intent

## Target Audience
Developers and engineers who need a fast, lightweight headless browser for web automation, scraping, testing, and AI agent workflows.

## Use Cases
- Setting up and running Lightpanda for headless web automation
- Integrating with Playwright, Puppeteer, or chromedp via CDP
- Building from source and contributing to the project
- Understanding the architecture and design decisions

## Documentation Focus
This documentation adopts a **technical guide** format:
- Quick start and installation instructions for all platforms
- CDP server setup and client integration examples
- Build from source prerequisites and steps
- Testing strategies (unit, E2E, Web Platform Tests)
- Architecture overview and design philosophy
- Contributing guidelines
